2. B.TECH Final year project


Forecasting Hydrocarbon production from Fractured wells



This project focuses on forecasting hydrocarbon production from fractured wells, which are crucial for extracting oil and gas from low-permeability reservoirs like shale formations. Unlike conventional reservoirs, fractured wells exhibit nonlinear flow behavior and transient production phases, making forecasting more complex.


The study compares two forecasting methods:
Decline Curve Analysis (DCA) A traditional empirical method using historical data to estimate production trends.
Machine Learning (LSTM) An advanced AI-based approach that dynamically learns complex production patterns.

The study concludes that while DCA is useful for general decline estimates, machine learning techniques like LSTM offer higher precision for unconventional reservoirs. Future forecasting techniques will likely integrate AI-driven models with real-time reservoir monitoring for better optimization.
